# Exploratory Analysis on Crime Records in Los Angeles

**UC Berkeley MIDS — DATASCI 200 Project 2**

---

## Project Overview

This project performs an exploratory data analysis of crime records in Los Angeles from 2020 to present, with a focus on understanding crime patterns across time, geography, and victim demographics. The analysis investigates whether COVID-19 and post-pandemic conditions influenced crime trends across different categories in LA.

---

## Data

Source: [LA City Open Data — Crime Data from 2020 to Present](https://catalog.data.gov/dataset/crime-data-from-2020-to-present)

The dataset includes ~800K+ crime incident records with fields covering crime type, date/time, location (area and coordinates), victim demographics, weapon used, and case status.

---

## Key Areas of Analysis

- Crime frequency trends over time (2020–present)
- Geographic distribution by LAPD area
- Victim demographics (age, sex)
- Crime type breakdown and weapon usage
- Data quality issues (missing fields, outliers, duplicates)
